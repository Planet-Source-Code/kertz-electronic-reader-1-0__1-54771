Whats New in Electronic Reader 1.0 ?

Electronic Reader is a free text-to-speech software. This program uses TruVoice Text-To-Speech Engine, so before you use this program you must install TruVoice Text-To-Speech Engine. It is provided with the software.

How To Use Electronic Reader 1.0 ?

Reading files using Electronic Reader is very easy. First you have to open a text file. Then click Begin Reading from the Control menu or from the toolbar. To change voice types click on the combo and select the voice type you wish.
